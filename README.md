# VirtualTreeView-Lazarus
This is the Lazarus port of Virtual Treeview control

## Requirements
Versions 4.x or 5.x
 * Lazarus 1.6 or newer
 * fpc 2.6.4 or newer
 * LCL Extensions 0.6 or newer
 
Version 6.x (lazarus_master branch)
 * Lazarus 1.6 or newer
 * fpc 3.1 (trunk) or newer
 * LCL Extensions 0.6 or newer
